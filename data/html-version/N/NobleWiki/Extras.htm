These are minor features of NobleWiki¬.

Minor ThingsDescription
Cancel buttonAdded to Edit page; basically onclick='window.location="&ScriptName()?$id;";'
Print Link ListUpdated this presentation to have a table. Any non-existent pages identified will show with the 'Create this entry' icon.
Recent ChangesInstead of $FS3 to split the recent changes log, use \W. For example ($ts) = split( /\W/, $fullrc[$i] );. Seems to work better for WIN32.
Page Lock IconAdded a small icon to the right of the page title to show that a page is locked or editable.
index.cgiThe wiki.pl script has been renamed.
IE6 textareaAdded <div style='width:100%;'>...</div> around the textarea in sub GetTextArea {} to enable 100% width in IE6
Table SortingAdded jQuery Tablesorter 2.0 to engage any table created properly. See WikiTables¬ for more information.
Disabled regex SearchA checkbox near the search form when marked will disable the default search mechanism that allows search terms to be interpreted as regex values.
Disable regex back-linksThe DoBackLinks subroutine no longer uses search terms as regular expressions.
Table Of ContentsThe Table of Contents may be assigned a pixel width parameter as in {{toc NNN}}
Expanded ConfigAdded a number of configuration variables.
Windows ConfigSet $Windows = 1 to remove CGITemp files after uploading assets.
Title attributeAdded a title attribute to all links where possible
Auto SignatureInsert [ [HomePage/RobertKurcina RobertKurcina] ] to automatically insert username with link to home page; defaults to 'Guest' and no link if not logged in. Use [ [HomePage/RobertKurcina RobertKurcina] at 2008.10.12 21:18 ] to append timestamp.
Bracketed Image URLs with options[URL upload:image|<options>] There's a number of options available now for images including title, description, boxclear, boxwidth, boxfloat, float, and clear. The last are for images within a caption box if description is provided. If description is provided but not title, then the title will show "FIGURE X" where X is computed as the current number of un-titled images
Edit BoxThe edit box and other textareas are always 100% width in every browser implementation.
Lock-out IndicatorsAdded small symbols to allow quick access to editing a page from the title, and symbols to indicate that a page is locked. '¬' is that symbol prefixed to a locked page.
Naming FilterProvide multiple keywords with or with negated terms to filter the names returned from backlinks, search, or the asset management interface.
